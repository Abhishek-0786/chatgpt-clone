# Connector Type Icons

This directory contains icon images for different connector types used in the Add Connector form.

## Required Images

Please add the following image files (PNG, SVG, or JPG format):

1. **type2.png** - Type 2 connector icon
2. **ccs2.png** - CCS 2 connector icon
3. **type1.png** - Type 1 connector icon
4. **gbt.png** - GB/T connector icon
5. **nacs.png** - NACS connector icon
6. **ac_socket.png** - AC Socket connector icon

## Image Specifications

- Recommended size: 64x64 pixels or larger (will be scaled down)
- Format: PNG with transparent background (preferred) or SVG
- The images will be displayed at 32px size in the connector type selection cards
