# Brand Logos Directory

This directory contains brand logo images for the vehicle selection dropdown.

## File Naming Convention

Please name the logo files according to the following pattern:
- Use lowercase letters
- Replace spaces with hyphens
- Use the exact brand name as shown in the mapping

## Image Format

- Supported formats: PNG, JPG, SVG
- Recommended size: 200x200px or larger (will be scaled down to 32x32px in dropdown)
- Background: Transparent (PNG) or white background
- Aspect ratio: Square or close to square for best results

## Notes

- If a logo file is missing, the system will automatically show a fallback with the brand's first letter
- Make sure file names match exactly (case-sensitive on some systems)
- The images will be automatically resized and displayed in the dropdown
