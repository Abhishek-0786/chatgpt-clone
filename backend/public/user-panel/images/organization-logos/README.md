# Organization Logos Directory

This directory contains logo images for different charging station organizations.

## Required Files:

1. **1c-ev-charging.png** (or .jpg) - Logo for "1C EV Charging" organization
2. **massive-mobility.png** (or .jpg) - Logo for "Massive Mobility" organization

## Notes:

- GenX organization uses the default robot icon (no image file needed)
- Supported formats: PNG, JPG, JPEG
- Recommended size: 40x40px to 60x60px for best display on map markers
- File naming must match exactly:
  - `1c-ev-charging.png` (or `.jpg`)
  - `massive-mobility.png` (or `.jpg`)

## Usage:

The logos will be automatically displayed on the map markers based on the station's organization.
